# Redux-2

Redux :-

- Redux is a Predictable state manageent tool. build using flux architecture.

Flux : -

- Action -> Dispatcher -> Reducer -> State -> View

Features of Redux :

- Predtable, Centralised, Easy Debugging.

Uses :-

- small :-
  - TodoApp
- Medium :-
  - Routing app,
  - company website
- Large :-
  - Authentication
  - Themeing
  - Product Management
  - Payment Management
  - etc.
